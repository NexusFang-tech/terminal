# Synthwave Resume Terminal

**Live:** [nexusfang-tech.github.io/terminal](https://nexusfang-tech.github.io/terminal)

Interactive retro terminal resume with a cold ops system persona. Features a full boot sequence with ASCII banner, command system for navigating skills, projects, experience, certifications, and contact information. CRT scanlines, retrowave perspective grid, command history, Tab autocomplete, and 15+ Easter eggs.

## Features

- **Boot sequence** — Simulated BIOS/kernel/auth/network initialization with progress bar
- **ASCII art banner** — Large "MATT KEITH" rendered in block characters
- **Command system** — `whoami`, `skills`, `projects`, `experience`, `certifications`, `contact`, `clear`, `help`
- **Skills visualization** — Capability matrix with filled/empty block progress bars and percentage ratings
- **Projects log** — Full operation log with tech tags and status badges
- **Tab autocomplete** — Start typing a command and hit Tab to complete
- **Command history** — Arrow up/down to navigate previous commands
- **15+ Easter eggs** — `sudo`, `hack`, `ls`, `pwd`, `nmap`, `vim`, `ping`, `ifconfig`, `cat /etc/operator_dossier`, `rm -rf /`, `matrix` (triggers Matrix-style character rain), and more
- **CRT effect** — Scanline overlay and vignette for authentic terminal feel
